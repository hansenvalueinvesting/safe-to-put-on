Instructions for expanding the database (when targeting specific brands):

1. Identify brand(s). 
2. Research to find all beauty/self-care products from this brand that are not currently in the products database. 
3. Add new products to the products database using database format.
4. Identify any new ingredients that are not covered in the ingredients database. 
5. Add new ingredients to the ingredients database using database format. When researching for citable data sources for ingredients, only use trusted regulatory sources + academic research.

Case-specific: if sorting for any entry works into a new category that isn't currently existing, build it out. Make sure these are broad categories though, nothing niche or specific. 
Do everything as efficiently as possible in terms of usage/compute resources; keep in mind database files may be massive in size too.
